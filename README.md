# 🌑 Shadow AI - WhatsApp Bot

بوت واتساب بالذكاء الاصطناعي مبني على Baileys
- **النصوص**: Gemini AI (بدون API key)
- **الصور**: NoteGPT + Catbox (بدون API key)
- **يشتغل فقط في المجموعات** التي البوت فيها **مشرف**

---

## 🚀 التشغيل

```bash
# 1. تثبيت المكتبات
npm install

# 2. تشغيل البوت
npm start
```

بعد التشغيل سيظهر **QR Code** في الطرفية.  
افتح واتساب ← الأجهزة المرتبطة ← امسح QR

---

## ✅ كيفية الاستخدام

1. أضف البوت لمجموعة واتساب
2. اجعله **مشرفاً** في المجموعة
3. أرسل أي سؤال وسيرد تلقائياً
4. أرسل صورة مع وصف وسيحللها

---

## 📁 هيكل المشروع

```
shadow-wa-bot/
├── index.js           ← الملف الرئيسي
├── package.json
├── plugins/
│   ├── gemini.js      ← Gemini AI للنصوص
│   └── analyze.js     ← NoteGPT للصور
├── sessions/          ← جلسة واتساب (تنشأ تلقائياً)
└── tmp/               ← ملفات مؤقتة (تنشأ تلقائياً)
```

## ⚠️ ملاحظات
- لا تحتاج أي API key
- البوت يتذكر آخر 30 محادثة
- الجلسة تُحفظ في مجلد `sessions/` ولا تحتاج QR مجدداً
